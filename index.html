Selected Topics - LLM Quiz

About LLMs

Large Language Models (LLMs) are AI systems trained on vast amounts of text data to understand and generate human-like text. They power many modern AI applications including chatbots, translation services, and content generation tools.

1. LLM stands for:

Correct Answer: a) Large Language Model

2. LLMs are mainly used for:

Correct Answer: b) Text generation

3. LLMs are trained on:

Correct Answer: a) Text data

4. Which model architecture is common in LLMs?

Correct Answer: c) Transformer
